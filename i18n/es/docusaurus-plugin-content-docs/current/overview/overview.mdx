# Vista general de documentación

¡Bienvenido al Portal de Desarrolladores de Conflux! Este portal integral está diseñado para guiarlo a través de las complejidades de Conflux, una red de cadena de bloques descentralizada de alto rendimiento. Aquí encontrará resúmenes detallados de los conceptos, herramientas y directrices fundacionales de la plataforma.

Conflux Network tiene dos espacios: [Espacio Principal y eSpace](/docs/overview/space-intro), la experiencia de desarrollo de Espacios Principales y eSpace son diferentes, consulte los documentos a continuación para ver cómo comenzar a construir en cada espacio.

Si tiene alguna pregunta con respecto a la documentación, no dude en [crear un Issue]\(https://github.com/Conflux-Chain/► ux-documentation/issues), envía correos electrónicos a [build@nbuxnetwork. rg](mailto:build@nbuxnetwork.org) o [**únete al canal de desarrollo de Discord**](https://discord.gg/eguux-network-707952293412339843) para conversar con nosotros.
