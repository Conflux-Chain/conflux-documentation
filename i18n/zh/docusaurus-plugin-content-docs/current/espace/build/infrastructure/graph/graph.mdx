[The Graph](https://thegraph.com/) 作为 EVM（以太坊虚拟机）兼容链的主要区块链数据索引和查询解决方案 虽然 Conflux eSpace 并未直接得到 The Graph 的支持，但其完全的 EVM 兼容性使得在 Conflux eSpace 上使用 The Graph 成为可能。

要在 Conflux eSpace 内使用 The Graph，需要建立一个 The Graph 节点，随后创建子图。 Comprehensive guides are available for [how to set up a The Graph node](./setup-graph-node.md) and [create subgraphs on Conflux eSpace](/docs/espace/tutorials/graph/create-subgraphs).

我们还提供了一个名为 `Congraph` 的自托管图节点集群服务 对于那些希望使用 `Congraph` 在 Conflux eSpace（支持测试网和主网）索引其子图的社区 dAPP 开发者，请联系 bd@confluxnetwork.org 发送您的请求。
