Conflux eSpace是一个独立的空间，它运行在与 [Core](../core/core.mdx)空间相同的基础设施上。 通过完全兼容的接口，以太坊的智能合约和dApps可以直接部署到eSpace，无需任何修改。 以太坊的开发工具、SDK、钱包和服务可以直接在eSpace中使用。

-   `Learn`: User guide of eSpace.
-   `Build`: introduces how to develop in eSpace.
