# Documentation Overview

欢迎来到 Conflux 网络开发者门户！ 这个全面的门户旨在帮助您理解 Conflux 网络的复杂性，一个高性能的去中心化区块链网络。 在这里，您将找到该平台的基础概念、工具和指南的详细总结。

Conflux 网络有两个空间：[Core Space 和 eSpace](/docs/overview/space-intro)，Core Space 和 eSpace 的开发体验不同，查看下面的文档了解如何开始在每个空间上构建。

If you have any questions regarding the documentation, feel free to [create an issue](https://github.com/Conflux-Chain/conflux-documentation/issues), send feedback emails to [build@confluxnetwork.org](mailto:build@confluxnetwork.org) or join our <DiscordLink>Discord channel</DiscordLink>  to engage in discussions with us.
